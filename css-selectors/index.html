Combination

  1. Descendant Selector:
    div a Anchor tag inside a Div
    Select all the elements inside of the first element Select anchors that are inside a div
  2. Direct Selector:
    div > span

    "p" inside a "div": Direct child

    Another direct span child
    - Select elements that are direct children of the first element. Select span that are direct children of a div
  3. General Sibling Selector:
    div ~ a

    sibling p tag

    - Selects all anchors that are siblings of a div and come after the div

  4. Adjacent Sibling Selector:
    div + a

    Adjacent sibling p tag

    - Selects all anchors that are siblings of a div and come directly after the div

  5. Or Selector:
    div, a

    p tag with class "first-p"

    p tag with class "second-p"

    - Selects all anchors and all divs

  6. And Selector:
    eg: div.c
    p.and

    p tag with class "and"

    p tag with class "and"

    - Selects all anchors and all divs
